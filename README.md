# Python_Blackjack_Game (Console Game)

A simple command-line Blackjack game written in Python.  
You play against the dealer using standard Blackjack rules.

## Features

- Full 52-card deck with suits and ranks
- Automatic handling of Ace as 1 or 11
- Dealer logic (hits until 17 or more)
- Supports playing multiple games in a row
- Clear text-based output showing your hand and the dealer's hand

## Requirements

- Python 3.8 or higher
- No external libraries required (uses only the Python standard library)

## How to Run
Make sure you have Python installed:
- python --version
